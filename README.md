# Sistema_Estacionamento_Projeto-Integrador
Solução de software para o gerenciamento e automação de estacionamentos, focada no controle de acessos, cálculo de tempo de permanência e tarifação de veículos, visando otimizar a gestão operacional e a precisão das cobranças, realizada no segundo semestre de 2025 no Projeto Integrador de Sistemas Computacionais do curso de ADS e SI da UNIEURO.
